Files containing the data

###########
chronic.txt : File containing the dynamics
t : time (year)
I : nutrient input
delta : fraction of direct recycling
d : decomposition rate
N : mineral nutrients
D : detritus
P : primary producers biomass
H : herbivore biomass
C : carnivore biomass
PP : primary producers production
HP : herbivore production
CP : carnivore production
PR : primary producers recycling
HR : herbivore recycling
CR : carnivore recycling
SP : secondary production
SR : secondary recycling
TP : total production
TR : total recycling


###########
bifurcation.txt : File containing the bifurcation from N to TR
I : nutrient input
delta : fraction of direct recycling
d : decomposition rate
value : value of the local extremum
variable : measured variable (N, D, P... TR)
