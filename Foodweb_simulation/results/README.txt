Files containing the data

###########
data.txt : File containing the general data of the simulation
nbSimu : ID of the simulation
I : nutrient input
delta : fraction of direct recycling
d : decomposition rate
NbSpeciesInit : initial number of species
NbPPInit : initial number of primary producers
NbSpeciesFinal : final number of surviving species
NbPPFinal : final number of surviving primary producers
PP : primary production
SP : secondary production
Irecy : average quantity of recycled nutrients
PPcv : coefficient of variation of primary production
PScv : coefficient of variation of secondary production
Irecycv : coefficient of variation of the quantity of recycled nutrients
TLmax : maximum trophic level in the food web
C : connectance
CV : average coefficient of variation of species
PR : primary recycling
SR : secondary recycling

###########
species.txt : File containing the average biomass of species
nbSimu : ID of the simulation
I : nutrient input
delta : fraction of direct recycling
d : decomposition rate
xi : biomass of species i

###########
CV.txt : File containing the coefficient of variation of species
nbSimu : ID of the simulation
I : nutrient input
delta : fraction of direct recycling
d : decomposition rate
xi : coefficient of variation of species i

###########
TL.txt : File containing the trophic level of species
nbSimu : ID of the simulation
I : nutrient input
delta : fraction of direct recycling
d : decomposition rate
xi : trophic level of species i
